# ng-ssr-cli

Angular2 server side rendering commandline client.

work with [angular2-universal](https://github.com/angular/universal).

## Try out

```
$ git clone git@github.com:vvakame/ng-ssr-cli.git
$ cd ng-ssr-cli
$ npm install

$ npm run build:example
$ npm start

$ npm run run-ngssr
.....!
```
